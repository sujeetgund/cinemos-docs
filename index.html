Be Updated with every Cinematic News

Cinemos API is an easy solution for films, songs or webseries related news database. It includes Bollywood, South-Indian and International films related news, including some of those you watch frequently.

Contribute 
const axios = require("axios");

const options = {
  method: 'GET',
  url: 'https://cinemos.p.rapidapi.com/news/bollywood',
  headers: {
    'X-RapidAPI-Key': 'YOUR_RAPIDAPI_KEY',
    'X-RapidAPI-Host': 'cinemos.p.rapidapi.com'
  }
};

axios.request(options).then(function (response) {
	console.log(response.data);
}).catch(function (error) {
	console.error(error);
});
